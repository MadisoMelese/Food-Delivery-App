# Food Delivery App

A full-stack food delivery application built to simulate a real-world food ordering platform. The app includes user authentication, menu browsing, cart and order management, and backend services for data handling and APIs.

---

## Tech Stack

**Frontend**
- React.js
- JavaScript / TypeScript
- Tailwind CSS / CSS
- Responsive design

**Backend**
- Node.js
- Express.js
- PostgreSQL / MongoDB
- RESTful API

---

## Features

**User & Session**
- User registration and login
- Authenticated routes
- Role-based access (user & admin)

**Ordering**
- Browse food items and categories
- Add items to cart
- Place orders
- View current and past orders

**Backend**
- Secure REST APIs
- CRUD operations for users, food items, and orders
- Database schema optimization
